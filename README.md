# hunzikerbaumpflege

Der HTML+CSS Code (simpler Code) für die Website hunzikerbaumpflege.ch steht in index.html. Er ist relativ einfach anpassbar, unverständliche Sachen können am besten von einer KI erklärt und angepasst werden. Nach bearbeiten und committen eines html Files oder uploaden eines Bildes kann es 30s dauern, bis die Website auch online die Änderungen übernimmt. Zur schnelleren Anpassung der Website ist es daher empfohlen, lokal das html file zu kopieren, mit einem Editor (ganz simpel: Notepad/Notizen, mit Hervorhebung durch Farben, Autovervollständigung etc. zb. VS Code), und das lokale html File neu zu laden.

Die Website hat bisher noch keine Subpage (zb. Hunzikerbaumpflege.ch/projekt-soundso) und ist "statisch", d.h. sie ist fix zu jeder Uhrzeit, für jeden Nutzer, usw. Eine Einbindung eines Kontaktformulares zb. bräuchte ein komplexeres Backend (Serverstruktur im Hintergrund) und wäre dann auch sicherheitstechnisch genau anzuschauen.
